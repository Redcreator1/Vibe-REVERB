Le pipeline

6 agents. Une mission. Ton PC.

🔍
Researcher
Structure l'enquête, identifie les gaps d'information, formule les hypothèses à tester. Ne ment pas sur les données — indique ce qu'il ne sait pas.
🧠
Strategist
Analyse positionnement, pricing, GTM et growth loops. Donne une recommandation claire avec plan 7 jours et métriques de succès.
💻
Coder
Propose architecture, APIs, data models et code propre. Suit les best practices du stack choisi, avec types et gestion d'erreurs.
⚡
Executor
Propose des commandes shell sûres et minimales. Ne s'exécute qu'avec ta validation explicite. Niveau de risque affiché pour chaque commande.
🔬
Critic
Revue indépendante de tous les outputs : contradictions, hypothèses non validées, risques d'exécution. Donne un Top 5 des corrections prioritaires.
✨
Synthesizer
Intègre tous les outputs en une recommandation cohérente, actionnable, avec plan 7 jours et métriques. C'est lui qui te livre le livrable final.

Questions fréquentes

Quel PC faut-il pour faire tourner AetherSwarm ?
Windows, Mac ou Linux. RAM minimum 8 Go (16 Go recommandé pour llama3.2:3b). Pas de GPU obligatoire — Ollama tourne en CPU. Sur Google Colab T4 (gratuit) c'est encore plus rapide.
Est-ce que j'ai besoin d'une clé OpenAI ou Anthropic ?
Non. AetherSwarm utilise Ollama par défaut (llama3.2:3b, gratuit, local). Si tu as une clé Anthropic Claude, tu peux la brancher pour des outputs plus puissants — mais c'est optionnel.
Mes données restent-elles sur ma machine ?
100%. Le backend tourne sur localhost:8787, les agents utilisent Ollama local, la mémoire est dans ChromaDB sur ton disque. Aucune donnée n'est envoyée à un serveur externe (sauf si tu actives Claude API volontairement).
Puis-je modifier les agents ?
Oui. Chaque agent a son fichier agents/<role>/prompts.py. Tu édites le prompt système, tu redémarres la console — les agents adoptent le nouveau comportement immédiatement.
Y a-t-il un support ?
Support par email : mindset22633@gmail.com. Le guide INSTALL.md et le CUSTOMIZE-AGENTS.md couvrent 95% des questions de démarrage.
