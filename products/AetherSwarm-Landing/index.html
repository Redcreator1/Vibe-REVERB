FAQ

Faut-il une connexion Internet pour utiliser Aether Swarm ?

Pour le téléchargement initial et pour installer Ollama, oui. Pour l'usage quotidien, non — tout tourne en local. Le toggle "Web DDG" est le seul appel sortant et il est optionnel.

Quel matériel pour faire tourner les agents ?

Minimum 8 Go de RAM, recommandé 16 Go. CPU récent (Ryzen / Intel 10e gen+). Un GPU NVIDIA accélère mais n'est pas requis. Les modèles 3B-7B tournent confortablement en CPU.

Est-ce que je peux modifier le code ?

Le code Python du backend et les prompts des agents sont fournis en clair. Tu peux les modifier pour ton usage. La redistribution ou la revente du code n'est pas autorisée.

Mac et Linux ?

Non, v1 est Windows uniquement.

Qu'est-ce qui se passe quand je ferme l'app ?

Le backend s'arrête proprement. Tes missions et ta mémoire Chroma sont sauvegardées sur disque. Au prochain lancement, tout est là.
