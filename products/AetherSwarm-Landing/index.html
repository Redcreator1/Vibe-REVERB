Questions fréquentes

De quoi j'ai besoin pour démarrer ?

Windows 10/11, Python 3.10+ (cocher "Add Python to PATH"), Ollama installé avec un modèle (`ollama pull llama3.2:3b`), 8 Go de RAM minimum. C'est tout.

C'est quoi exactement Ollama ?

Ollama est un outil gratuit qui fait tourner des LLMs (Llama, Mistral, Qwen...) directement sur ton PC. Pas de clé API, pas de frais. AetherSwarm l'utilise par défaut.

Est-ce que mes données partent sur Internet ?

Non. Par défaut tout tourne en local. Tu peux activer la recherche web (DuckDuckGo) ou Claude API volontairement si tu veux — mais c'est ton choix, pas celui du produit.

Est-ce que je peux modifier le code ?

Oui. Le code Python est livré en clair, modifiable pour ton usage personnel et commercial. Pas de redistribution ni de revente du code.
