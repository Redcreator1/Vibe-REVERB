# Mission 03 — Analyse concurrentielle profonde

**Mode recommandé :** Swarm + toggle Web DDG activé  
**Durée estimée :** 6-12 min  
**Quand l'utiliser :** Avant de fixer ton prix, ton positionnement ou de lancer une nouvelle feature.

---

## La mission

```
Active le toggle Web DDG.

Je veux une analyse concurrentielle complète de mon marché.
Mon produit : [DESCRIPTION]
Mes concurrents connus : [C1, C2, C3] (laisse vide si tu veux que les agents les trouvent)

Analyse :
1. Cartographie les 5 acteurs principaux : pricing, modèle de revenus, segment ciblé, point faible
2. Identifie les 3 "jobs to be done" non couverts ou mal couverts par les concurrents
3. Trouve mon angle différenciant le plus défendable (difficile à copier en < 3 mois)
4. Propose un positionnement en 1 phrase qui me place hors de la comparaison directe
5. Liste les 5 signaux de marché (avis négatifs, Reddit, forums) qui confirment l'opportunité
```

---

## Variante "prix" :
```
Analyse les modèles de pricing de [C1, C2, C3].
Identifie où je peux me positionner pour maximiser la valeur perçue sans guerre de prix.
Recommande un prix précis avec justification psychologique et économique.
```
