# Mission 02 — Copywriting Landing Page complète

**Mode recommandé :** Swarm complet  
**Durée estimée :** 5-10 min  
**Quand l'utiliser :** Tu veux une landing page qui convertit, écrite par 6 agents qui se critiquent mutuellement.

---

## La mission

```
Mon produit : [NOM DU PRODUIT]
Ce qu'il fait : [DESCRIPTION EN 1 PHRASE]
Pour qui : [AUDIENCE]
Prix : [PRIX]
Mes 3 plus gros concurrents : [C1, C2, C3]
Ma différence principale : [DIFFÉRENCIATEUR]

Rédige la landing page complète au format copywriting professionnel :
1. Hero : headline percutant + sous-titre + CTA
2. Section "Problème" : agitation émotionnelle en 3 points
3. Section "Solution" : comment mon produit résout chaque point
4. Section "Features" : 6 fonctionnalités avec bénéfice associé (pas juste la feature)
5. Section "Social proof" : 3 témoignages fictifs réalistes à remplacer par les vrais
6. Section FAQ : 5 objections fréquentes + réponses
7. CTA final avec garantie

Ton : [professionnel / décontracté / urgence / autorité] — choisis le plus adapté à l'audience.
```

---

## Variantes

**Version "headline A/B test" :**
```
Génère 10 headlines pour [PRODUIT] ciblant [AUDIENCE]. 
Varie les angles : curiosité, douleur, résultat, chiffre, question, contre-intuitif.
Classe-les par potentiel de conversion estimé.
```

**Version "email de lancement" :**
```
Rédige une séquence de 3 emails de lancement pour [PRODUIT] :
Email 1 (J-7) : teaser + problème
Email 2 (J-1) : solution + preuves
Email 3 (J0) : offre + urgence + CTA
```
