# Mission 07 — Cold outreach & acquisition clients (sans budget pub)

**Mode recommandé :** Swarm complet  
**Durée estimée :** 5-8 min

---

## La mission

```
Mon produit : [PRODUIT]
Ma cible : [PROFIL CLIENT IDÉAL — ex: "freelance développeur React, 2-5 ans d'expérience"]
Mon canal : [LinkedIn / email / Twitter DM / forum]
Objectif : [10 premiers clients / 50 beta users / 5 partenaires]

Construis ma machine d'acquisition organique :
1. Profil ICP ultra-précis : où ils trainent, ce qu'ils lisent, leurs douleurs top-of-mind
2. Séquence cold outreach en 3 messages (accroche, valeur, relance) — prêts à copier-coller
3. 10 endroits précis où aller les trouver cette semaine (communautés, hashtags, forums, events)
4. Script de 1er appel découverte : 5 questions qui qualifient et vendent sans vendre
5. Objection "je n'ai pas le budget" → réponse mot pour mot
```

---

## Variante "partenariats" :
```
Je veux trouver 3 partenaires qui ont déjà mon audience cible pour faire des promos croisées.
Mon produit : [PRODUIT]. Audience : [AUDIENCE].
Identifie les types de partenaires idéaux, comment les approcher, et rédige le message d'intro.
```
