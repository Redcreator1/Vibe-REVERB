# Mission 04 — Plan de contenu 30 jours (LinkedIn / X / Newsletter)

**Mode recommandé :** Strategist seul (plus rapide) ou Swarm  
**Durée estimée :** 3-6 min  
**Quand l'utiliser :** Tu veux construire une audience avant ou pendant ton lancement.

---

## La mission

```
Je suis solopreneur. Mon domaine d'expertise : [DOMAINE].
Mon produit / service : [PRODUIT].
Mes plateformes cibles : [LinkedIn / X / Newsletter / YouTube — choisis].
Mon audience : [QUI].
Objectif 30 jours : [notoriété / leads / ventes / communauté].

Génère un plan de contenu complet pour 30 jours :
1. 4 piliers éditoriaux adaptés à mon expertise et mon audience (avec ratio de publication)
2. Calendrier semaine par semaine : 1 post/jour avec angle + hook d'accroche + format (texte / carousel / thread / vidéo)
3. 5 posts "cornerstone" à fort potentiel viral : structure complète prête à publier
4. Stratégie de repurposing : comment 1 post devient 3 formats différents
5. KPIs à suivre semaine par semaine pour ajuster le plan
```

---

## Variante "post viral LinkedIn" :
```
Rédige un post LinkedIn viral sur le sujet [SUJET] pour une audience de [AUDIENCE].
Format : hook en 2 lignes + corps en bullet points + CTA.
Longueur : 150-200 mots max. Ton : [authentique / éducatif / provocation douce].
```

## Variante "thread X" :
```
Rédige un thread X de 10 tweets sur [SUJET].
Tweet 1 = hook fort (chiffre ou contre-intuitif).
Tweets 2-9 = insights actionnables.
Tweet 10 = CTA vers [LIEN / PRODUIT / NEWSLETTER].
```
