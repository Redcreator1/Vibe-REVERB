## Comment utiliser ce pack

1. **Lance AetherSwarm** (`start-aether.bat`)
2. **Ouvre la mission qui correspond à ton besoin du jour**
3. **Copie le bloc de mission** dans le composer de la console
4. **Remplis les crochets** `[COMME CECI]` avec tes infos
5. **Active Mode Swarm** (et optionnellement Web DDG ou Memory)
6. **Lance et lis les 6 agents travailler pour toi**

---

## Tips pour maximiser les résultats

- **Sois précis dans les crochets** — plus tu donnes de contexte, meilleure est la sortie
- **Active Memory** sur les missions stratégiques — tes insights s'accumulent session après session
- **Active Web DDG** sur les missions qui nécessitent des données de marché (missions 01, 03, 07)
- **Utilise les variantes** — chaque mission a des versions courtes et spécialisées
- **Enchaîne les missions** — ex: fais la 03 (concurrents) avant la 02 (landing page)
